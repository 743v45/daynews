OpenJDK 发布 JEP 540，为 Java 平台引入一个简单的 JSON API（孵化阶段）。该 API 旨在无需外部库即可解析和生成符合 RFC 8259 的 JSON 文档，强调低仪式感、小体积和易用性，但不支持数据绑定、流式处理或语法扩展。
