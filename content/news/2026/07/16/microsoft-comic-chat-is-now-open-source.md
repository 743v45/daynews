微软宣布开源 Microsoft Comic Chat——1996 年发布的 IRC 聊天客户端，它能把对话自动转化为漫画形式（带对话框和表情），也是 Comic Sans 字体的首发载体。源代码现已在 GitHub 上公开，包含现代化的构建尝试。
