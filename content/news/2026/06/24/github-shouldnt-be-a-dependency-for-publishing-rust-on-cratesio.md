crates.io 当前仅支持 GitHub 账号登录和发布包，导致平台过度依赖 GitHub，存在单点故障和访问受限问题。社区呼吁引入更多登录方式，但受限于技术债务、反垃圾能力不足等问题，进展缓慢。
