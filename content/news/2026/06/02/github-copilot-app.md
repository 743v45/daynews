GitHub 推出 Copilot 桌面应用技术预览版，支持从 Issue 到合并的完整开发生命周期。可并行运行多个隔离的 Agent 会话，集成 MCP 服务器和自定义技能，实现自动化工作流。目前向 Copilot Pro/Pro+/Max/Business/Enterprise 用户开放。
