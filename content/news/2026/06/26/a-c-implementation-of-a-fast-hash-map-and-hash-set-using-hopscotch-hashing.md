Tessil 在 GitHub 上开源了 hopscotch-map 库，这是一个基于 hopscotch hashing 和开放寻址法的 C++ 哈希映射与哈希集合实现。它是仅头文件的库，性能优于 std::unordered_map，内存占用小于 google::dense_hash_map，并支持异构查找等功能。
